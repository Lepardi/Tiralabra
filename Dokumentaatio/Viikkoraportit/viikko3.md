# Viikkoraportti 3

Viikolla projektiin käytetyt tunnit: ~20t

1. Mitä tehty tällä viikolla: \
Projektiin kirjoittu testejä tekoälyn funktioida testaamaan. Aikaa meni myös sopivien testien miettimiseen ja testaamiseen. \
Kokeilin eri tapoja karsia expectiminimaxin satunnaisten tapahtumien huomioonottoa jotta tekoälyn pelinopeutta saisi kasvatettua. Ohjaajan kanssa puunasimme heuristiikkaa sekä tekoälyn koodia ja tulimme tulokseen että kannattaa pidättäytyä kaikki tapahtumat läpikäyvässä expectiminimax mallissa vaikka sillä hakusyvyyden kasvattaminen onkin tällä hetkellä haastavaa koska se lisää peliaikaa huomattavasti. 

2. Projektin edistyminen: \
Projekti on hyvällä mallilla ja aikalailla dokumentointia sekä kattavampaa testaamista vailla. Tekoälyn peliteho "puhtaalla" expectiminimaxilla tehostui heuristiikkojen puunamisella. Peliaika myös parantui jokusen verran pelin ja tekoälyn koodia puhdistamalla. Tässä olisi vieläkin kuitenkin parannettavaa.

3. Mitä opin tällä viikolla: \
Sain ohjaajalta hyviä vinkkejä python ohjelmien tehokkuuteen vaikuttavista tekijöistä.

4. Mitä jäi epäselväksi: \
Tällä hetkellä asiat ovat aikalailla selvät.

5. Mitä seuraavaksi: \
Kattavien testien miettiämistä ja toteuttamista.
